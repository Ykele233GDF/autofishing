# AutoFishing

Automatically detects bobber splashes and reels in fish with precise timing. Toggle on/off with **K** key. Fully client-side.

## Features

- Detects fishing bobber movement and auto-reels at the right moment
- Auto-cast after reeling in a fish
- Toggle on/off with K key (status shown on action bar)
- Works with any fishing rod
